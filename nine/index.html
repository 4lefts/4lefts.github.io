Nine

An FM drone synthesizer.

I like the idea of an instrument that can be played by moving faders or dials. I wanted to use html <input>s for this, and learn how to style them. Thanks to the excellent css-tricks for this.

Obviously this is a beginning, and there are many more obvious things to add. Modulation sources are one, so I’ll be looking into which parameters it makes sense to modulate using an LFO, and which it doesn’t. The other obvious issue is pitch; there’s no way to have any real control over what note the synth plays. But then, it is for drones.

Built using the p5js library and p5js.sound.
